# Jonathan Broqueza Portfolio

Interactive personal portfolio website for Jonathan Broqueza, a BSCS graduate from Bicol University Polangui.

## What this website presents

- Custom websites
- Landing pages
- Booking systems
- Management systems
- Portfolio websites
- Full-stack website development
- UI/UX design based on client requirements

## Main sections

- Interactive hero section
- Selected projects
- Services
- Development process
- About section
- Tools and technologies
- Contact section
- Admin page for managing projects locally

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Vercel-ready deployment

## Admin note

The `/admin` page currently saves projects to browser local storage. This is useful for quick editing and previewing project lists. For a production admin system shared across devices, connect it later to Supabase, Firebase, or another database with authentication.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
